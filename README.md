📋 CAD-PRO – Sistema de Cadastro Web

Aplicação web desenvolvida em PHP com integração a banco de dados relacional, voltada para o gerenciamento de registros e cadastros.

🚀 Funcionalidades
Cadastro de usuários/registros
Edição e exclusão de dados
Listagem de informações
Persistência em banco de dados
Operações CRUD completas
🛠️ Tecnologias utilizadas
PHP
HTML/CSS
Banco de dados relacional (MySQL ou outro)
SQL
Git/GitHub
📂 Estrutura do projeto
/config → Conexão com banco de dados
/models → Representação dos dados
/controllers → Lógica da aplicação
/views → Interface do usuário

(adapte se sua estrutura for diferente)

▶️ Como executar
Clone o repositório:
git clone https://github.com/CarlosESoares/cadpro-php-system
Configure o banco de dados
Execute em um servidor local (XAMPP, WAMP ou similar)
